# stock_system_good_us_v2 — 에이전트 지침

## 일일 자동 스캔 루틴: 요약(이메일/푸시) 포맷

스케줄된 일일 스캔 루틴의 마지막 한국어 요약은 사용자의 이메일로 발송되며,
GitHub-flavored Markdown으로 렌더링된다. **표로 표현 가능한 항목은 반드시
마크다운 표로 작성한다.** 구체적으로:

1. **엘리트픽**: 아래 컬럼의 마크다운 표로 작성 (불릿 목록 금지).

   | 순위 | 티커 | 이름 | 가격 | 합산점수 | 급등확률 |
   |---|---|---|---|---|---|

   엘리트픽이 없으면 "엘리트픽 없음"이라고 명시.

2. **전체 후보 TOP 5**: 엘리트픽 표와 겹치지 않는 정보가 있을 때만 별도 표로,
   겹치면 "TOP 5 = 엘리트픽 상위 5개 (티커 나열)" 한 줄로 처리.

3. **tracker 실측 적중률(+10%/10일)**: 집계돼 있으면 표
   (표본수 / 적중수 / 적중률)로, 없으면 한 줄로 명시.

4. 표 외 항목(스캔 기준일, 후보 수, 데이터 수집 실패 종목, 리스크 한 줄)은
   짧은 프로즈/불릿로 유지한다. 리포트 전문은 붙여넣지 않는다.
